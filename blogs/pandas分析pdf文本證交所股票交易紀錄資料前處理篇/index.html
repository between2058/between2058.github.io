pandas分析PDF文本證交所股票交易紀錄(資料前處理篇)

Johnny Chang | Feb 27, 2020

在這個漸漸邁向大數據的時代裡，資料將會成為最珍貴的寶物，為了要快速用程式分析這些大數據，我們不但要將未來的各種事物直接導向數位化，更要學習如何把過去的傳統紙本資料轉化為數位格式。

前言

先簡單描述一下我們的原始資料(raw data):

總共有15份pdf檔，每份約200頁，約共3000頁。 每個pdf檔裡面含有約50個人的證交所成交紀錄表。 本專案一共有將近600個人。 試想 : 若今天我想知道某某某在2013年11月12號中，一共買進多少股票，該怎麼做?

你可以告訴我 : 「很簡單啊，找到某某某的交易紀錄，按一按計算機就好了。」

但，

如果某某某在2013年11月12號共有500筆交易紀錄呢 ? 如果我想知道其他天的呢 ? 如果我想統計這一個月以來的一共買進賣出多少股票呢 ? 有太多太多如果了，人工方式按計算機是行不通的，何不將他製做成一個資料庫呢 ?

開始任務

定下目標

我想把這將近600人的臺灣證交所交易紀錄表，製作成pandas的dataframe，方便之後可以使用pandas提供的優秀函式庫來做資料分析。

發現問題，打算怎麼做?

首先要思考電腦要如何消化這一大疊的資料套表。

  1. 有沒有pdf電子檔 ? 如果沒有，趕快去掃描。
  2. 有了pdf電子檔以後，裡面的文字是不是能夠複製的那種(反白)? 如果不是，趕快去想辦法文字辨識(可參考我的文章)。
  3. 假設此pdf電子檔有內嵌文字，那麼就有跡可循了。

資料前處理

這次的例子比較幸運，有pdf電子檔也有內嵌文字，因此我二話不說先做了一個小實驗 ，直接把pdf內容反白複製貼上到一個記事本裡面 ，假設我覺得內容的格式很適合利用python做字串處理，那就採用這種好笑的方法。 看了下圖以後，第一個想到的就是.split()，第二個想到的是他剛好每一筆資料就是記事本的一行，可以利用txt讀檔的角度來處理，在前處理的過程中應該能夠減輕不少煩惱。(ps. 雖然這個方法很好笑，但是我認為現實問題沒有標準解法，只有比較符合需求的解法)

因此，馬上來把這15份pdf檔的內容每一頁通通複製貼上在同一個記事本裡面，取名為 AB_type_text.txt 存下來，然後試著用python讀取它 :

with open(r"C:\Users\Johnny\Desktop\python\AB_type_text.txt", 'r',encoding="utf-8") as infile:
    for i in infile:
        print(i)

接著讓我來專注在我們想要的臺灣證券交易所成交資料表上面。 我先假設這15個pdf檔格式非常整齊統一，例如每個人的格式都一樣的情況下，我才可以設立條件來做全盤的篩選出我要交易資料表。 所以我先來觀察其中的一筆資料:

觀察一 :

好不好 .split() ?

str = "01 20060602 xxxx 6950 0034780 F311233333 陳卻B 0 09132272 000086439 60344 002015 000002000 0000040300 N 1380 0"
# 以空白格切割字串
str = str.split(" ")
print(str)

output:

結果 : split()效果很好，同時已將這一行字串轉為一個list了。

觀察二 :

這個 list 長度有多長 ?

print(len(str))
====================================================================
output:
17

觀察三 :

長度為17可以做為證交所交易資料的篩選條件嗎 ?

with open(r"C:\Users\Johnny\Desktop\python\AB_type_text.txt", 'r',encoding="utf-8") as infile:
    for i in infile:
        i = i.split(" ")
        if len(i) == 17:
            print(i)
====================================================================
output:
['01', '20060602', 'xxxx', '6950', '0034780', 'F311233333', '陳卻B', '0', '09132272', '000086439', '60344', '002015', '000002000', '0000040300', 'N', '1380', '0']
['0000440', '1060013978', 'F311233333', '0950113', '1060517', '證保法字第１０６０００１６９０號財團法人證券投資人及期貨交易人保護中心93020196215', '陳卻1010204', '1010206', '706', '融資買進1234', 'ㄚㄚㄚ阿華永台中+', '000000000000000', '+', '000000001000000', '+', '000000001000000', '融資\n']

結果:光是使用長度條件來篩選會不小心篩選到其他長度也是為17的資料。 然而，證交所的第二個元素是日期，且為%Y%m%d的8碼格式，而另外一筆資料的第二個元素的字串長度並不是8，所以我們再多設立一個條件試試看。

觀察四 :

長度為17 且 第二個元素字串長度為8 有用嗎?

with open(r"C:\Users\Johnny\Desktop\python\AB_type_text.txt", 'r',encoding="utf-8") as infile:
    for i in infile:
        i = i.split(" ")
        if len(i) == 17 and len(i[1])==8:
            print(i)
====================================================================
['01', '20060602', 'xxxx', '6950', '0034780', 'F311233333', '陳卻B', '0', '09132272', '000086439', '60344', '002015', '000002000', '0000040300', 'N', '1380', '0']

結果: 恭喜成功，我們成功只篩選出證交所資料囉~ 美中不足的是，姓名後面寫了一個B，其實那是一個非常有意義的資料，那代表著這一筆資料是BUY，反之S則是SELL。因此我們最好把這個有用的資訊趕快跟姓名隔開。

觀察五 :

把B和姓名隔開。

with open(r"C:\Users\Johnny\Desktop\python\AB_type_text.txt", 'r',encoding="utf-8") as infile:
    for i in infile:
        i = i.split(" ")
        if len(i) == 17 and len(i[1])==8:
            i.insert(7,i[6][-1])
            i[6] = i[6][:-1]
            print(i)
====================================================================
output:
['01', '20060602', 'xxxx', '6950', '0034780', 'F311233333', '陳卻','B', '0', '09132272', '000086439', '60344', '002015', '000002000', '0000040300', 'N', '1380', '0']

結果: 恭喜阿恭喜，我們完成初步的資料前處理了~

轉成Dataframe

按照觀察出來的步驟，我們很快的可以篩選出這15份PDF檔中，每一個人的每一筆證交所成交紀錄的 list。 接下來我們所要做的事就是把每一筆 list 接住，因此我們可以創建一個空的list在開頭，取名為result。之後每traversal一筆資料，就將其append到result裡頭(list包list的概念)

有了list包list的概念以後，很輕易就可以做成pandas的dataframe，操作如下:

import pandas as pd
df = pd.DataFrame(result)

接著可以把不想要的欄位去掉，有兩種做法:

  1. 把不要的去掉 ，用del
  2. 把要的留著，類似條件式的篩選，也就是我的作法，如下:
col = [1,2,3,5,6,7,9,10,12,13,14]
df = df[col]
df

還可以替欄位重新命名，做法如下:

df.columns = ["成交日期","股票代碼","證商帳號","身分證字號","姓名","買賣別","成交時間","成交單價","成交股數","成交金額"]
df

還可以日期作為index，做法如下:

df = df.set_index("成交日期")
df

我們得到了這樣的Dataframe :

dataframe 轉 CSV

最後我們簡單的把這個Dataframe存成一個csv檔，作為保存。

df.to_csv("證交所交易紀錄表.csv",encoding="utf-8-sig")

若日後還想用python讀取此csv檔案的話，也可以這麼做 :

df.read_csv("證交所交易紀錄表.csv", encoding="utf-8-sig", engine="python")
